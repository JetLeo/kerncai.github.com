编辑~/.inputrc（没有的话，就新建一个），在最后加一行：

{% highlight sh %} set completion-ignore-case on {% endhighlight %}

保存，重新打开终端即可。这样，TAB键补全的时候，就会忽略大小写了，非常方便快捷

用方向键Up，Down来搜索以该串字符开头的历史命令。

在~/.inputrc下面添加

{% highlight sh %} “\e[A”: history-search-backward “\e[B”: history-search-forward {% endhighlight %}

或者在~/.bashrc中加入：

{% highlight sh %} bind '"\e[A": history-search-backward' bind '"\e[B": history-search-forward' {% endhighlight %}

比如，bash下 输入d，然后上下键，就会匹配出d相关的命令了
